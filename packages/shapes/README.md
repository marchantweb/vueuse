# @vueuse/shapes

[![NPM version](https://img.shields.io/npm/v/@vueuse/shapes?color=a1b858)](https://www.npmjs.com/package/@vueuse/shapes)

## Install

<pre class='language-bash'>
npm i <b>@vueuse/shapes</b> @vueuse/core
</pre>
